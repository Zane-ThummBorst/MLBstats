About the project

This is just a small side project From me (Zane ThummBorst) To learn and pick up Django.I wanted to make a project related to something I was casually interested in and found a nice python Package that acted as a wrapper to a baseball API. I have a couple of ideas to continue on based on what the package is capable of, and I am currently working on adding bootstrap to the rest of the pages/ make everything look consistent in the frontend. This project also makes use of smaller techs like JQuery, whitenoise, and JSON files(to be converted into a noSQL DB).
