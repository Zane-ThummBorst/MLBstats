About the project

This is just a small side project From me (Zane ThummBorst) To learn and pick up Django.I wanted to make a project related to something I was casually interested in and found a nice python Package that acted as a wrapper to a baseball API. I have a couple of ideas to continue on based on what the package is capable of, and I'm also considering looking into combining the project into a React frontend (but that's going to be painful at this point so ¯\_(ツ)_/¯). The project also makes use of smaller techs like JQuery, whitenoise, and a JSON pseudo DB (just a file full of names but super useful).
